Alta de Empleados con PowerAutomate

Este proyecto surge de la necesidad de automatizar el proceso de alta de empleados en la empresa.

Con PowerAutomate se crea un flujo que permite recibir un correo cuando un empleado rellena un formulario, procesar sus datos y decidir automáticamente si generar un usuario y contraseña, asignar un ordenador o simplemente registrar la información sin acción adicional.

El objetivo es ahorrar tiempo y errores, integrando Recursos Humanos en un mismo flujo automatizado, mostrando cómo las herramientas de Microsoft pueden agilizar procesos administrativos.

Cómo visualizar y utilizar el proyecto

Este proyecto se presenta en formato video, donde se muestra el flujo de PowerAutomate y su funcionamiento. No requiere instalación de software adicional más allá de Microsoft PowerAutomate y Forms.

En el video se muestra:
- El formulario el cual tiene que rellenar el empleado.
- La recepción de correo automático con los datos del empleado.
- Los condicionales para decidir asignar ordenador, generar usuario y contraseña.
- Las notificaciones automáticas al área de Recursos Humanos.
