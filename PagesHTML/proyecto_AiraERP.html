Aira ERP

Aira ERP es una aplicación desarrollada completamente en Python que gestiona compras, ventas, almacén, estadísticas y tablas maestras utilizadas en la empresa. También cuenta con una página web de presentación y venta del software.

Funcionalidades Destacadas

Aira ERP permite gestionar de forma integral los procesos clave de cualquier empresa, con interfaces de usuario intuitivas y reportes estadísticos precisos.

Compras

Gestión completa de pedidos de proveedores, control de stock y registro de facturas de compra.

Ventas

Registro de pedidos de clientes, generación de facturas y control de cobros y pagos.

Almacén

Control de inventario, movimientos de stock, entradas y salidas de productos.

Estadísticas

Informes y gráficos para análisis de ventas, compras y rendimiento del negocio.

Usuarios

Creación de Usuarios con los datos de dicha persona.

Instalación y ejecución local

Aira ERP requiere Python y MySQL para funcionar correctamente. Si deseas probarlo en tu ordenador, puedes hacerlo utilizando XAMPP para la base de datos y ejecutando el proyecto en local.

A continuación, los pasos recomendados para instalarlo y ejecutarlo:

  1. Descargar e instalar XAMPP.
  2. Ejecutar XAMPP y encender Apache y MySQL.
  3. Configurar la base de datos:

    • Opción 1: Importar la base de datos incluida en el proyecto (aira_erp.sql) desde phpMyAdmin.
    • Opción 2: Crear tu propia base de datos desde cero, definiendo tus tablas y relaciones según las necesidades de tu empresa.
  4. Editar el archivo conexion.py con los datos de tu servidor local (host, usuario, contraseña, base de datos).
  5. Ejecutar la aplicación de una de estas dos formas:

    • Desde Python con el comando: python main.py
    • Abriendo directamente el ejecutable AiraERP.exe incluido en el proyecto
  6. ¡Listo! Ya puedes comenzar a gestionar compras, ventas, almacén y estadísticas desde Aira ERP.
