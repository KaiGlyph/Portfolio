Anikai

Anikai es una plataforma web dedicada al mundo del anime, diseñada tanto para organizar colecciones personales como para permitir que la comunidad anime interactúe y comparta sus series favoritas. Desarrollada con React, TypeScript y desplegada en Vercel.

Funcionalidades Destacadas

  • Catálogo de Anime: Base de datos completa con información detallada de series, películas y OVAs del mundo del anime.
  • Sistema de Favoritos: Guardar y organizar los animes favoritos de cada usuario, con listas personalizadas.
  • Mi Lista: Seguimiento del progreso de visualización: pendiente, en curso, completado, en pausa, abandonado.
  • Comunidad: Espacio para reseñas, recomendaciones y discusiones entre fans del anime.
  • Búsqueda y Filtrado Avanzado: Por género, año, estudio, estado, puntuación y más criterios.
  • Sistema de Valoraciones: Puntuaciones y reseñas de usuarios para descubrir nuevas series.
  • Recomendaciones Personalizadas: Basadas en preferencias y listas del usuario.

Secciones Principales

Anikai está organizado en varias secciones para ofrecer una experiencia completa a los fans del anime:

  • Explorar: Descubre nuevos animes por género, popularidad, estreno reciente o recomendaciones.
  • Mi Lista: Organiza tus animes en diferentes estados: viendo, completado, pendiente, en pausa, abandonado.
  • Favoritos: Colección personal de tus animes preferidos para acceso rápido.
  • Comunidad: Foro de discusiones, reseñas y recomendaciones entre usuarios.
  • Perfil: Estadísticas personales, historial de actividad y configuración de preferencias.
  • Búsqueda: Motor de búsqueda avanzado con múltiples filtros y opciones de ordenación.

Tecnologías Utilizadas

Anikai está construido con las siguientes tecnologías:

  • React 18: Biblioteca JavaScript para interfaces de usuario interactivas y dinámicas.
  • TypeScript: Tipado estático para mayor seguridad y mantenibilidad del código.
  • Vite: Build tool moderno y rápido para desarrollo frontend optimizado.
  • Vercel: Plataforma de hosting y despliegue para aplicaciones web modernas.
  • Tailwind CSS: Framework CSS utility-first para diseño rápido y consistente.
  • React Router DOM: Sistema de enrutamiento para navegación entre secciones.
  • Framer Motion: Animaciones fluidas y transiciones para mejorar la UX.
  • Local Storage API: Almacenamiento de datos del usuario en el navegador.

Estado del Proyecto

En Desarrollo Activo

Anikai se encuentra en fase inicial de desarrollo. Se está trabajando en la estructura base, el diseño de la interfaz y la integración con la base de datos para almacenar la información de anime y usuarios.

Próximos Hitos:

  • Diseño y desarrollo de la interfaz principal del catálogo
  • Implementación del sistema de búsqueda y filtrado
  • Creación del sistema de favoritos y listas personales
  • Integración con base de datos para almacenamiento de datos
  • Desarrollo de la sección de comunidad y reseñas
  • Optimización para dispositivos móviles
  • Sistema de autenticación de usuarios
  • Panel de estadísticas personales
  • Pruebas de usabilidad y rendimiento

Inspiración:

Anikai está inspirado en plataformas como MyAnimeList y AniList, pero con un enfoque en la simplicidad, el diseño moderno y la experiencia de usuario intuitiva. El objetivo es crear una plataforma que combine la funcionalidad de las grandes plataformas con una interfaz más accesible y atractiva visualmente.
