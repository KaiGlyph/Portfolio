FairyPaws

Fairy Paws surge de la tarea que tenía para hacer un inicio de sesión el cual, si no estás iniciado, te restringe algún campo. En este caso, está hecho para que inicies sesión o no podrás hacer una donación a los pandas rojos (todo de forma ficticia).

Para añadir un poco más de complejidad, estética y funcionalidad, decidí hacer *modals* para el inicio y el registro de sesión, de forma que no te mande a otra página para luego tener que volver.

Como tenía tiempo para probar cosas nuevas, decidí añadir unas partículas en la pasarela de pago, de forma que diera una estética de cerezo en flor.

Para finalizar, ya que el inicio de sesión era sencillo porque no podía poner más campos (ya que era una base de datos para el examen), añadí condiciones para tener una contraseña más segura.

Y con todo esto me quedo conforme con el resultado en el tiempo que tuve para crearlo, que fueron unas dos semanas o menos, ya que ayudaba a los compañeros de clase con problemas que tenían.

Instalación y ejecución local

Esta página es más compleja de abrir y observar el proceso de inicio de sesión, ya que se necesita MySQL, además de que se utilizan archivos .php. Esto impide abrir la página directamente desde GitHub, como normalmente lo hago.

Por lo tanto, se deben seguir los siguientes pasos para ejecutarla correctamente de forma local:

  1. Descargar e instalar XAMPP.
  2. Abrir la ruta: C:\xampp\htdocs\
  3. Copiar dentro esa carpeta el proyecto deseado (en mi caso: PaginasWebs\FairyPaws).
  4. Ejecutar XAMPP y encender Apache y MySQL.
  5. Abrir la base de datos (opcional) para ver los datos disponibles.
  6. Abrir un navegador y escribir: http://localhost/PaginasWebs/FairyPaws/
  7. ¡Listo! Ya puedes probar el inicio de sesión.
